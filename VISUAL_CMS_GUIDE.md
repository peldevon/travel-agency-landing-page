## üó∫Ô∏è Managing Tours

### Viewing All Tours

1. Click **"View All Tours"** from the dashboard
2. Browse through all your tour packages
3. Use the search bar to find specific tours
4. Filter by status (Published/Draft)

### Creating a New Tour

1. Click **"Add New Tour"** button
2. Fill in the required information:

#### Basic Information
- **Tour Title**: e.g., "Weekend in Dubai"
- **URL Slug**: Auto-generated, but customizable
- **Category**: Choose Local, Regional, or International
- **Status**: Draft or Published
- **Duration**: Number of days and nights
- **Pricing**: Set prices in NGN and USD
- **Short Description**: Brief summary for card previews
- **Full Description**: Detailed tour description (WYSIWYG editor)
- **Seasonality**: Best time to visit

#### Highlights
- Click **"Add Highlight"** to add key features
- Example: "Burj Khalifa tour", "Desert safari"
- Add as many highlights as needed
- Remove unwanted highlights with the trash icon

#### Itinerary
- Click **"Add Day"** to add each day of the tour
- For each day, provide:
  - **Day Title**: e.g., "Arrival & City Tour"
  - **Description**: Detailed activities for that day
- Reorder or remove days as needed

#### Inclusions & Exclusions
- **Inclusions**: What's included in the price
  - Example: "Round-trip flights", "4-star hotel"
- **Exclusions**: What's NOT included
  - Example: "Travel insurance", "Personal expenses"

#### Gallery Images
- Add image URLs for the tour photo gallery
- First image will be the main/hero image
- Recommended: 4-6 high-quality images
- Use Unsplash or upload to your own image hosting

#### Additional Information
- **Visa Notes**: Visa requirements and support
- **Cancellation Terms**: Refund and cancellation policy
- **WhatsApp Prefill**: Pre-written message for enquiries

3. Click **"Create Tour"** to save

### Editing an Existing Tour

1. Go to the Tours management page
2. Click **"Edit"** on the tour you want to modify
3. Make your changes
4. Click **"Save Changes"**

### Deleting a Tour

1. Click the **trash icon** on the tour card
2. Confirm deletion in the popup
3. The tour will be permanently removed

## üé® Using the WYSIWYG Editor

The Rich Text Editor (WYSIWYG - What You See Is What You Get) allows you to format text without knowing HTML.

### Basic Formatting
- **Bold**: Click **B** or Ctrl+B
- **Italic**: Click *I* or Ctrl+I
- **Underline**: Click U or Ctrl+U
- **Headings**: Use dropdown to select H1, H2, H3
- **Lists**: Create bulleted or numbered lists
- **Links**: Highlight text ‚Üí Click link icon ‚Üí Enter URL

### Advanced Features
- **Align Text**: Left, center, right, justify
- **Text Color**: Change text and background colors
- **Insert Images**: Add images inline
- **Blockquotes**: Highlight important quotes
- **Code Blocks**: For technical content

### Tips
- Use **H2** for main sections
- Use **H3** for subsections
- Keep paragraphs short (3-4 lines max)
- Add **bold text** for emphasis
- Use **bulleted lists** for easy scanning

## üí° Best Practices

### Content Writing Tips

1. **Be Specific**: Use concrete details instead of vague descriptions
2. **Use Active Voice**: "Explore ancient ruins" vs "Ancient ruins can be explored"
3. **Include Numbers**: "Visit 3 UNESCO World Heritage Sites"
4. **Answer Questions**: What's included? What should I bring? What's the best time?
5. **Add Personality**: Write in a friendly, conversational tone

### Tour Package Tips

- **Clear Pricing**: Always state what's included/excluded
- **Realistic Itinerary**: Don't overpack the schedule
- **Highlight Unique Features**: What makes this tour special?
- **Update Seasonality**: Keep "best time to visit" current
- **Test WhatsApp Links**: Make sure enquiry messages work

### Shortlet Property Tips

- **Accurate Photos**: Show the actual property, not stock images
- **Complete Amenities List**: List everything to avoid surprises
- **Clear House Rules**: Set expectations upfront
- **Responsive Descriptions**: Mention WiFi speed, workspace quality for remote workers
- **Update Availability**: Mark properties as "booked" when reserved

## üö® Troubleshooting

### Can't Login
- **Solution**: Check username/password spelling
- **Clear browser cache** and try again
- Contact your web administrator if issues persist

### Changes Not Showing on Website
- **Hard refresh** the website: Ctrl+F5 (Windows) or Cmd+Shift+R (Mac)
- Wait 1-2 minutes for cache to clear
- Check if content status is "Published" not "Draft"

### Images Not Displaying
- Check if image URL is correct (starts with http:// or https://)
- Make sure image URL is publicly accessible
- Try opening the image URL in a new browser tab

### Lost Unsaved Changes
- The CMS does not auto-save
- **Always click Save** before navigating away
- Consider writing long content in a text editor first, then pasting
